# Quick Start (3 steps)

## 1. Setup (one-time)
```bash
./setup.sh
```

## 2. Configure API Key
Edit `.env` and add your GLM API key:
```bash
GLM_API_KEY=your_actual_api_key_here
```

Get your API key from: https://open.bigmodel.cn/

## 3. Run
**Terminal 1** (Backend):
```bash
./start_backend.sh
```

**Terminal 2** (Frontend):
```bash
./start_frontend.sh
```
